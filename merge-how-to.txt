Чтобы получить ветку с сервера, надо  написать:
git fetch

Для слияния веток прописываем поочерёдно:
1. переключиться в ветку друга, чтобы создать её локально: git checkout surname
2. посмотреть все ветки: git branch --all
3. переключиться в основную ветку: git checkout master
4. обьединить ветку друга с основной веткой: git merge surname

После слияния веток: 
1. отправить изменения на сервер: git push
2. посмотреть историю репозиория: git log
